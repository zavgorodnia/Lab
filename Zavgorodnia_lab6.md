## 6.  Знайти кількість значень < 0 для кожного стовпця
```
less0 <- function(mat) {
  col <- ncol(mat)
  for (i in 1:col) {
    col[i] <- sum(mat[,i]<0)
  }
  col
}
less0(mat)

```

```
[1] 5 6 6 5 7
```
